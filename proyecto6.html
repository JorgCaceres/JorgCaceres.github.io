Sobre el proyecto

Bash 𐤟 Logs 𐤟 Servidor remoto 𐤟 SQL

En este proyecto se realizaron búsquedas de información de errores y solicitudes en logs y se trabajó haciendo consultas a una base de datos remota. A través de PostgresSQL en línea de comandos.

La información requerida en los informes de logs era para encontrar las solicitudes de direcciones IP específicas, encontrar errores en cierto rango de horarios con códigos específicos. Y guardar los registros encontrados creando carpetas y archivos con la información a través de líneas de comando. Para las consultas de base de datos, lo que me requirieron fue realizar consultas a una base de datos de una aplicación de gestión de taxis, donde entre otras cosas, me requirieron consultar. La cantidad de vehículos y su información, verificar los costos según el clima, etc. Las fotos y los comandos utilizados, tanto para la consulta de logs, como para las consultas SQL, se encuentran en los enlaces a las evidencias que están más abajo.
