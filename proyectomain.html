Sobre el proyecto

Diseño de pruebas 𐤟 Caja Negra 𐤟 Postman 𐤟 AndroidStudio 𐤟 JIRA

En este proyecto, realicé pruebas a un conjunto de aplicaciones integradas llamadas Urban Scooter, una aplicación de renta de scooters en línea. Esta incluye una aplicación web para los usuarios que rentan los scooters y una aplicación móvil para los usuarios que realizan las entregas de estos scooters. Este es el proyecto más completo que me ha tocado realizar como QA Tester, ya que no solo realizamos pruebas a los distintos módulos de la aplicación 'Urban Scooter', sino que también llevamos a cabo el análisis de requerimientos y el diseño de las pruebas adecuadas para verificar los requisitos solicitados.

En este proyecto se aplicaron técnicas de visualización y análisis de pruebas de caja negra, como la creación de mapas mentales. Se descompusieron los requisitos a probar y se agruparon en clases de equivalencia, enfocando las pruebas en los valores límites. Además, se elaboraron listas de comprobaciones, casos de prueba, informes de errores, entre otros procesos. Todo el proceso de testing fue aplicado tanto para la aplicación web como para la aplicación móvil. Para la aplicación móvil, se utilizó Android Studio para facilitar el cumplimiento de los requisitos especificados, verificando su conformidad. Además, se trabajó con Postman para realizar pruebas al backend mediante solicitudes HTTP, asegurando el cumplimiento del formato de los headers y el body especificados en la APIdoc

Toda la información detallada se puede revisar en la evidencia adjunta más abajo, donde incluyo enlaces a todos los documentos que fueron creados durante el proceso de pruebas de estas aplicaciones.
