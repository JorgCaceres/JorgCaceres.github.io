Sobre el proyecto

Postman 𐤟 JSON 𐤟 JIRA 𐤟 API REST

En este proyecto se realizaron pruebas a una nueva función agregada por el equipo de desarrollo a la API Urban.Grocers, aplicación que gestiona los usuarios, productos y kits. De un programa para realizar compras en supermercados y almacenes asociados.

Utilizando Postman, se probó la capacidad de agregar productos a los kits que ofrecen los almacenes. La disponibilidad de servicios de entregas adheridospermitió que se probaran distintas funciones en el módulo de carrito, a través de solicitudes POST, GET, PUT y DELETE realizadas con Postman. Con sus headers y body correspondientes según las especificaciones en la APIdoc y el documento de requerimientos. Para estas pruebas Se prestó especial atención a los códigos de respuesta y verificación de requisitos.

La lista de comprobaciones resultante constaba de más de 60 pruebas con sus respectivos informes de errores en JIRA.
