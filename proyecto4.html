Sobre el proyecto

Pytest 𐤟 Request

En este proyecto se realizaron pruebas automatizadas a la API Urban.Grocers, aplicación que gestiona los usuarios, productos y kits. De un programa para realizar compras en supermercados y almacenes asociados.

Se nos entregó una lista de comprobación con las pruebas a realizar, donde verificamos que se estuvieran cumpliendo los requerimientos de valores límite, entre otros. De los campos para la creación de kits, comprobando a través de Pytest que el código de la respuesta coincida con los de la documentación. Con código Python y a través de la librería Requests, se ejecutaron solicitudes POST. Para obtener autorización y luego poder crear los Kits con sus headers y body correspondientes según las especificaciones en la APIdoc y el documento de requerimientos.

El código de las pruebas se puede obtener en los enlaces a la evidencia que se encuentran más abajo.
