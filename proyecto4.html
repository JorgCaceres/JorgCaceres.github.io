Sobre el proyecto

En este proyecto se realizaron pruebas automatizadas a la API Urban.Grocers, aplicacion que gestiona los usuarios, productos y kits de un programa para realizar compras en supermercados y almacenes asociados.

Se nos entrego una lista de comprobacion con las pruebas a realizar, donde verificamos que se estuvieran cumpliendo los requerimientos de valores limites, entre otros de los campos para la creacion de Kits, comprobando a traves de Pytest que el codigo de la respuesta coincida con los de la documentacion. Con codigo Python y a traves de la libreria Requests, se ejecutaron solicitudes POST para obtener autorizacion y luego poder crear los Kits con sus headers y body correspondientes segun las especificaciones en la APIdoc y el documento de requerimientos.

El codigo de las pruebas se puede obtener en los enlaces a las evidencia que se encuentran mas abajo.
