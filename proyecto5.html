Sobre el proyecto

Python 𐤟 Pytest 𐤟 Selenium WebDriver 𐤟 POM

En este proyecto se realizaron pruebas automatizadas a la aplicación web Urban Routes, aplicación de transporte de pasajeros, en específico a la función de pedir un taxi.

Se nos entregó una lista de comprobación con las pruebas a realizar, donde verificamos el correcto funcionamiento de las funciones de configurar la dirección de partida y de llegada, seleccionar. La tarifa correcta registra números de teléfono y licencias, entre otras funciones adicionales dependiendo de la tarifa seleccionada, cumpliendo los requerimientos de cada uno de los formularios. Que se deben completar para realizar este escenario de prueba. Se siguió el paradigma de la OPP y el patrón de diseño POM para realizar las pruebas, separando los selectores, las pruebas y las funciones en distintas páginas. Todas escritas utilizando Python y Selenium WebDriver.

El código de las pruebas se puede obtener en los enlaces a la evidencia que se encuentran más abajo.
